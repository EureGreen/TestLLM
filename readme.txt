Запускаются 2 варианта
-telegramBot.py 	\\Вариант с телеграм бота, отвечает при обращении к username
-chat.py 	\\Консольный вариант


Запуск:
-Открываем CMD
-Переходим в директорию с приложением
-Запускаем командой Python telegramBot.py\chat.py




env.
OPENROUTER_API_KEY= \\Вставляете api с openrout желательно с выводом текста;
ALLOWED_USERS= \\Разрешенные пользователи для бота;
TELEGRAM_BOT_TOKEN= \\Если запускать вариант бота в телеграм, нужен апи бота;
